# HTTP 常见的状态码有哪些

HTTP 常见的状态码及其含义如下：

```shell
1xx（信息性状态码）：服务器收到请求正在处理。

2xx（成功状态码）：请求已成功被服务器接收、理解、并接受。
200 OK：请求成功，返回的信息在响应体中。
204 No Content：请求成功，但是响应体为空。

3xx（重定向状态码）：需要客户端进一步操作来完成请求。
301 Moved Permanently：资源的 URI 已被更新，客户端应使用新 URI 发送请求。
302 Found：资源的 URI 临时被更新，客户端应继续使用原有 URI 发送请求。

4xx（客户端错误状态码）：客户端请求出错。
400 Bad Request：请求存在语法错误。
401 Unauthorized：请求需要用户验证。
404 Not Found：请求的资源不存在。

5xx（服务器错误状态码）：服务器出错。
500 Internal Server Error：服务器发生了错误，无法完成请求。
503 Service Unavailable：服务器处于维护或过载状态，暂时无法处理请求。
```
